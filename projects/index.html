My Process

First, I created the "project1" folder and put the subfolders "audio" and "images" to store its respective files. Next, I copied a Bootstrap template from W3Schools to work with. Then, I planned the webpages for the website. Finally, I actually created the website.

Decisions

  1. To make a separate folder for each type of file that I would use.
  2. To choose the "Webpage" Bootstrap template from W3Schools.
  3. To include the same logo and footer text on each page.
  4. To have a section for links for every page except the home pages.
  5. To make my webpages about my hobbies and favorite trap/pop songs, respectively.
  6. To have a section for each separate hobby/song on each page.
  7. To put the Google Map on the home page.

Template Modifications

  • Removed a link in the navbar for a login section.
  • Removed a 2 column section that was for ads, since I wasn't going to put any ads on the page.
  • Made the other two sections (2 column and 8 column, respectively) increase their columns by one (3 columns and 9 columns) to take up the space of the one deleted.
  • Changed the sans-serif font to a serif font from Google Fonts.

The Easy Parts

To be honest, most of it was things I've already learned the year before and had done recently, or was something I easily understood or had in mind, were very easy. I already knew how to set Google fonts, make each heading type, and include audio into a webpage. This made making the website very easy, for the most part.

The Hard Parts

The only hard part was the making of the Google map. As I had no prior experience with it, it took me about half an hour to fully understand how I was supposed to include markers and a polygon using those markers. Even then, it took a bit of work and looking back and forth in order to make the code work and display with little difficulty.
